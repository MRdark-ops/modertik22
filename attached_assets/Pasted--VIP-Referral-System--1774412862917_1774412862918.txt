أريد إنشاء نظام عمولات متدرج احترافي (VIP Referral System) داخل منصة تداول، بنفس أسلوب العرض البصري والهيكل التالي:

🎯 أولاً: واجهة المستخدم (UI Design)
- تصميم عصري بخلفية داكنة (Dark Theme)
- استخدام ألوان:
  - أزرق + ذهبي + أخضر (للأرباح)
- إضافة عناصر:
  - أيقونات (أشخاص – مخططات – أرباح – عالم)
  - تأثيرات Glow و Neon
  - عرض المستويات في بطاقات (Cards)
- كل مستوى يظهر في Box منفصل يحتوي:
  - اسم المستوى (Level 1 → Level 5)
  - عدد الأشخاص المطلوب
  - نسبة العمولة
  - الربح لكل شخص
  - إجمالي الأرباح

⚙️ ثالثاً: شروط تفعيل العمولة (IMPORTANT LOGIC)
- لا يتم احتساب الإحالة مباشرة
- يجب على المستخدم المحال:
  1. التسجيل عبر رابط الإحالة
  2. القيام بعملية Deposit
  3. رفع إثبات الدفع (Proof Upload)
- يتم إرسال الطلب إلى Admin Dashboard
- يقوم الأدمن:
  - Accept / Reject
- فقط عند Accept:
  - يتم تفعيل الإحالة
  - يتم إضافة العمولة

🔐 رابعاً: نظام التتبع (Tracking System)
- لكل مستخدم:
  - Referral ID
  - Referral Link
- تتبع:
  - عدد الإحالات لكل مستوى
  - الأرباح لكل مستوى
  - إجمالي الأرباح
- Dashboard يحتوي:
  - Earnings
  - Levels Progress Bar
  - Active / Pending Referrals

📊 خامساً: مميزات إضافية
- Progress Bar لكل مستوى
- نظام إشعارات:
  - عند قبول الإحالة
  - عند كسب عمولة
- Anti-Fraud System:
  - منع الحسابات الوهمية
  - منع التكرار (Duplicate Accounts)

🚀 الهدف:
إنشاء نظام احترافي يشجع المستخدمين على بناء فريق (Network) وتحقيق أرباح تصاعدية بطريقة واضحة وجذابة بصرياً.
